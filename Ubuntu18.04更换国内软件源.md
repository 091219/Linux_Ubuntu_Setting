



## 以Ubuntu18.04为例：

1. 备份原有软件源文件

   ```
   cp /etc/apt/sources.list /etc/apt/sources.list.bak_yyyymmdd
   ```

2. #### 获取Ubuntu代号

   lsb_release -a       (18.04 为bionic)

3. #### 打开sources.list文件，在文件末尾增加如下软件源地址

   ```
   vi /etc/apt/sources.list
   ```

4. 追加如下地址：
